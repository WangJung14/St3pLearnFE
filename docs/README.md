# docs/README.md - Bộ tài liệu triển khai Frontend

Thư mục này chứa tài liệu hướng dẫn triển khai phần Frontend cho hệ thống **St3p-Learn**.

## Danh sách tài liệu

```txt
docs/
├─ 01-frontend-scope.md
├─ 02-routing-and-pages.md
├─ 03-api-integration.md
├─ 04-auth-rbac.md
├─ 05-state-management.md
├─ 06-ui-components.md
├─ 07-implementation-roadmap.md
└─ 08-manual-test-checklist.md
```

## Thứ tự đọc đề xuất

1. Đọc `Design.md` ở thư mục gốc để nắm kiến trúc tổng thể.
2. Đọc `Agents.md` nếu dùng AI Agent/Cursor/Codex để code FE.
3. Đọc `01-frontend-scope.md` để biết phạm vi làm.
4. Đọc `02-routing-and-pages.md` để dựng route.
5. Đọc `03-api-integration.md` và `04-auth-rbac.md` trước khi code API.
6. Đọc `07-implementation-roadmap.md` để chia task theo phase.
7. Dùng `08-manual-test-checklist.md` để test trước khi demo.
